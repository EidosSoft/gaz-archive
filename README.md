# Gaz Archive Manager

Инструмент для создания и управления архивами с расширением `.gaz` (ZIP‑архивы с кастомным расширением).

## Возможности

* Создание новых `.gaz`-архивов из файлов и папок
* Добавление файлов в существующие архивы
* Извлечение содержимого архивов
* Просмотр списка файлов внутри архива

## Установка

```bash
git clone https://github.com/ваш-username/gaz-archive.git
cd gaz-archive
pip install -r requirements.txt
```

## Использование

Запустите интерактивный менеджер:
```bash
python gaz_manager.py
```

## Примеры команд

1. **Создать новый архив:** выберите `1`, укажите имя архива и путь к папке/файлу.
2. **Добавить файл:** выберите `2`, укажите имя архива и полный путь к файлу.
3. **Извлечь содержимое:** выберите `4`, укажите имя архива и папку для извлечения.

## Требования


* Python 3.6+
* Стандартные библиотеки Python (`zipfile`, `os`, `pathlib`)
